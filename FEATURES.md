# Claude Code Usage Monitor - 功能总览

## 🎯 核心功能

### 1. 实时监控 ✨
- **总体统计卡片**
  - 总花费（USD）
  - 总 Tokens 使用量
  - 总请求次数
  - 平均每次请求成本

- **自动刷新**
  - 每 30 秒自动更新数据
  - 手动刷新按钮
  - 实时显示最后更新时间

### 2. 预算管理 💰
- **预算设置**
  - 每日预算限额
  - 每周预算限额
  - 每月预算限额
  - 自定义警告阈值（默认 80%）

- **预算监控**
  - 可视化进度条显示
  - 实时计算使用百分比
  - 超过阈值自动警告
  - 颜色编码（绿色→橙色→红色）

- **警告提醒**
  - 页面顶部显示警告横幅
  - 详细说明当前花费和限额
  - 多级警告支持

### 3. 数据可视化 📊
- **花费趋势图**
  - 折线图显示花费变化
  - 双 Y 轴：花费 + 请求次数
  - 支持多种时间粒度（小时/天/周/月）
  - 交互式图表（悬停显示详情）

- **模型分布图**
  - 饼图显示各模型花费占比
  - 彩色区分不同模型
  - 悬停显示具体金额

### 4. 详细统计 📈
- **模型统计表**
  - 每个模型的详细数据
  - 请求次数
  - 输入/输出/缓存 Tokens
  - 总花费
  - 排序和筛选

- **最近请求记录**
  - 最近 20 条请求
  - 时间戳
  - 使用的模型
  - Token 数量
  - 单次花费

### 5. 时间范围筛选 ⏰
- 最近 24 小时
- 最近 7 天
- 最近 30 天
- 所有时间
- 动态调整统计粒度

### 6. CLI 命令行工具 🖥️
- **快速查询**
  ```bash
  python monitor.py              # 所有统计
  python monitor.py -t 24        # 最近24小时
  python monitor.py -r           # 最近请求
  python monitor.py -r -n 20     # 最近20条
  ```

- **输出格式**
  - 格式化的表格
  - 彩色输出（终端支持）
  - 易于阅读的布局
