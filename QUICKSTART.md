# 快速开始指南

## 方法一：使用启动脚本（推荐）

```bash
# 1. 运行启动脚本
./start.sh

# 2. 在浏览器中打开
open http://localhost:5000
```

## 方法二：手动启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
python app.py

# 3. 访问监控面板
# 浏览器打开 http://localhost:5000
```

## 方法三：使用命令行工具

```bash
# 查看所有统计
python monitor.py

# 查看最近24小时
python monitor.py -t 24

# 查看最近请求
python monitor.py -r

# 查看帮助
python monitor.py -h
```

## 功能测试

### 1. 查看主面板
- 访问 http://localhost:5000
- 查看总体统计卡片
- 观察图表是否正常显示

### 2. 测试预算功能
- 点击"⚙️ 设置预算"
- 输入预算限额
- 保存并查看预算进度条

### 3. 切换时间范围
- 使用页面顶部的时间选择器
- 选择不同的时间范围
- 观察数据更新

### 4. 刷新数据
- 点击"🔄 刷新"按钮
- 或等待自动刷新（30秒）

## 常见问题

**Q: 没有数据显示？**
A: 确保你已经使用过 Claude Code 并产生了一些请求。尝试选择"所有时间"时间范围。

**Q: 端口被占用？**
A: 修改 app.py 中的端口号，或者关闭占用 5000 端口的其他应用。

**Q: 图表不显示？**
A: 检查浏览器控制台是否有 JavaScript 错误，确保已正确加载 Chart.js。
